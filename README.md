## Environment Variables

To run this project, you will need to add the following environment variables to your `.env` file

`NODE_ENV`
`DB_DATABASE`
`DB_HOST`
`DB_USERNAME`
`DB_PASSWORD`
`DB_DIALECT`
`JWT_SECRET`
`JWT_EXPIRE`
`JWT_COOKIE_EXPIRE`
`GG_CLIENT_ID`
`GG_CLIENT_SECRET`
`GG_REFRESH_TOKEN`
`GG_REDIRECT_URI`


## Start Project

To run this project

```bash
  npm run start
```
